  • Primeiro Projeto

    Foi o primeiro contato com o profissional do desenvolvimento de software, onde eu fazia parte da equipe de 4 desenvolvedores. O projeto consistia em um sistema de treinamentos para empresas arquitetado em micro serviços. No começo, eu fiquei criando testes unitários e de integração e, após isso, trabalhei nas entidades e value objects.

    Após esse tempo de adaptação, recebi algumas responsabilidades e fiz a implementação de alguns conteúdos e adquiri muitos conhecimentos sobre APIs REST.

  • VueJS

    Este projeto foi desenvolvido com o intuito de aprender e praticar os conceitos do Vue.js, um framework JavaScript progressivo para a construção de interfaces de usuário. A simulação de uma lanchonete foi escolhida como cenário para aplicar esses conhecimentos. O projeto utiliza o json-server, uma biblioteca que permite criar um servidor RESTful com base em um arquivo JSON. Isso nos permite simular um back-end.

  • Ride

    Esse projeto foi um teste que eu realizei onde a API deveria possibilitar aos usuários a criação de pedais ciclísticos para que outros usuários possam visualizá-los e se inscrever. O projeto foi desenvolvido em Typescript, GraphQL, Express, Prisma, Postgres e Docker.

    Este foi um desafio muito grande, pois eu não conhecia bem o GraphQL, mas com muita pesquisa e estudo consegui realizar o projeto e foi muito satisfatório ver o projeto funcionando.
